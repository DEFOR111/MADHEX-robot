# The Hexapod 
**MADHEX -** is fully Open sourse project for those who wanna build their hexapod 

----------------------------------------------------------------------------------------------------------------------------------------------------

**the main features :**
* 18 DOF Allows walk, rotate at one place , and even climb
* High stability
* can walk almost any terrain
* looks great

----------------------------------------------------------------------------------------------------------------------------------

## PCB 
<img width="410" height="330" alt="PCB Side view" src="https://github.com/user-attachments/assets/6529c81a-d61f-4e7d-b465-47b9cd992bae" />

**Include** : 
* 40A and up to 600W of power on Buck converters  which provide stabily and wont burn while the peak of the robot servos
* 2 power lines 1 for 9 servos which provide great performance(J5-J20 is A-servo_power line J21-J36 is B-servo_power line it is important you should use 9 servos for each power line for better perfomnace and power symetry )
* 2 INA226 sensors which allows to track the power and get Data
* 32 PWM ports for those who wanna add additional stuff into robot
* BNO055 IMU which provide great perfomance and stability
* Esp32-s3-Wroom-1 MCU for inverse kinematics and great perfomance
* 2 PCA9685 which can give up to 32PWN signals
* AMS1117 3.3V for logic
* Usb-c port
* 2smd buttons 

**Overall of PCB -** the pcb of the robot is great one because it is really small compact and can handle 32PWM signals which makes it a way more customizable
